## Introducción
Mecanográfico es un minijuego para practicar tu velocidad de escritura desde la terminal hecho con Java 17.

Si descargas el repo y estás desde Linux o Mac sólo debes ejecutar `MECANOGRAFICO-linux-mac.sh`.

Si por otro lado te encuentras en Windows tán solo debes ejecutar `MECANOGRAFICO-windows.bat`.

En ambos casos es necesario tener Java 17 o + instalado.
